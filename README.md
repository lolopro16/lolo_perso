## Pong

Un petit clone du jeu Pong réalisé en HTML5/Canvas.

### Lancer le jeu

1. Ouvrez le fichier `pong.html` dans votre navigateur préféré.
2. Utilisez les touches `W` et `S` pour contrôler la raquette de gauche.
3. Utilisez les flèches `↑` et `↓` pour contrôler la raquette de droite.

Amusez-vous bien !
